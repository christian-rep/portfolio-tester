# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Estructura base del proyecto con HTML, CSS y JavaScript
- Barra de navegación fija con enlaces ancla
- Sección "Quien Soy" con presentación personal
- Sección "Formación Académica" con listas de estudios
- Sección "Experiencia Laboral" con tabla responsiva
- Sección "Intereses" con iconos representativos
- Sección "Contacto" con formulario validado
- Footer con íconos de redes sociales y efectos hover
- Diseño completamente responsive con media queries
- Navegación suave entre secciones
- Validación de formulario en tiempo real
- Animaciones al scroll
- Atributos de testabilidad (data-testid)
- Documentación completa (README, TESTING-SPECS, DEPLOYMENT)
- Licencia MIT

### Technical Features
- HTML5 semántico y accesible
- CSS3 con Flexbox y Grid
- JavaScript vanilla ES6+
- Sin dependencias externas
- Optimizado para performance
- Soporte para accesibilidad (ARIA)
- Estructura preparada para testing

### Fixed
- Ajustes de márgenes y paddings
- Corrección de colores según especificaciones
- Optimización de imágenes y iconos
- Mejoras en la experiencia móvil
